# Laboratorio de TICS
## Proyecto final IoT
Proyecto final laboratorio de TICS haciendo uso de sensor de sonido KY-038 

## Componentes a utilizar
- Raspberry Pi
- Sensor Sonido KY-038
- Arduino UNO 
- Cable Arduino USB tipo AB
- Cables Jumper
- Breadboard
- Led Roja
- Resistencia 220  $\ohm$
